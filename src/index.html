ReactIRC

ReactIRC is a flask-inspired library for developing IRC bots. It comes with an embedded web server for receiving POST requests from webhook enabled services.


Install

Git

$ git clone \
http://git.citruspi.io/citruspi/ReactIRC.git
$ cd ReactIRC
$ python setup.py install

PyPi

$ pip install reactirc


Examples

Greeter Bot

Greeter Bot listens for messages which match the pattern Hey, my name is (name) and replies by greeting the user with Hey there (name)!. 
#!/usr/bin/python
from ReactIRC import Bot

bot = Bot()

@bot.on(r'^Hey, my name is (.*)$')
def greeter(name):

    return 'Hey there %s!' % name

bot.monitor(nick='bot', channels='#bots')

Karma Bot

Karma Bot manages user karma. User's can ++ and -- each other to increase and decrease karma. Simply entering karma will return your karma count, and adding a user's name to the end (karma citruspi) will return the karma count for the targetted user. 
#!/usr/bin/python
from ReactIRC import Bot

bot = Bot()

scores = {}

@bot.on(r'^karma$')
@bot.on(r'^karma ([\S]+)$')
def karma_status(user=None):

    if user not in scores.keys():

        return '%s has 0 karma' % user

    else:

        return '%s has %s karma' % (user, scores[user])

@bot.on(r'^([\S]+)(\+\+|\-\-){1}$')
def karma_adjustment(user, direction):

    if user not in scores.keys():

        scores[user] = 0

    if direction == '--':

        scores[user] -= 1

    elif direction == '++':

        scores[user] += 1

bot.monitor(nick='bot', channels='#bots')
